                    快速部署步骤
================================================================================

方式一：使用 Python（推荐）
---------------------------------
1. 解压文件：
   unzip release0930.zip -d acne-website
   cd acne-website

2. 启动服务器：
   python3 -m http.server 8080

3. 访问网站：
   http://localhost:8080


方式二：使用 Nginx
---------------------------------
1. 解压文件到 Web 根目录：
   unzip release0930.zip -d /var/www/html/acne

2. 配置 Nginx（可选）：
   server {
       listen 80;
       server_name your-domain.com;
       root /var/www/html/acne;
       index index.html;

       location / {
           try_files $uri $uri/ =404;
       }
   }

3. 重启 Nginx：
   sudo systemctl restart nginx


方式三：使用 Apache
---------------------------------
1. 解压文件到 Web 根目录：
   unzip release0930.zip -d /var/www/html/acne

2. 配置 .htaccess（已包含在压缩包中，如需自定义）：
   <IfModule mod_rewrite.c>
       RewriteEngine On
       RewriteBase /acne/
       RewriteCond %{REQUEST_FILENAME} !-f
       RewriteCond %{REQUEST_FILENAME} !-d
       RewriteRule ^.*$ index.html [L]
   </IfModule>

3. 重启 Apache：
   sudo systemctl restart apache2


方式四：部署到云服务
